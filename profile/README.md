### 🛠 Tech Stack

- **Frontend**: Next.js 14 (App Router) + React
- **Backend**: Next.js API Routes
- **Search**: Custom Index + Crawler Scrapper
- **AI**: Cribble AI + Grounded Answers
- **Rewards & Mining**: On-chain integration (Search-to-Earn + Search-to-Mine)
- **EdTech**: CribEdu platform
- **Weather**: Open-Meteo
- **Geo**: Nominatim
- **Analytics**: Supabase
